# 沉浸式翻译接入指南

沉浸式翻译（Immersive Translate）是一款优秀的双语对照翻译扩展，支持网页、PDF、EPub 等多种格式的翻译。通过配置 ZenMux，您可以使用我们聚合的高质量翻译模型来提升翻译体验。

## 配置步骤

### 1. 获取 ZenMux API Key

前往 [ZenMux 控制台](https://zenmux.ai/console) 注册账户并获取您的 API Key。

::: tip 快速开始
仅需几分钟即可完成配置，立即体验高质量的多模型翻译服务
:::

### 2. 安装沉浸式翻译

1. 在浏览器扩展商店搜索"沉浸式翻译"
2. 安装扩展并启用
3. 打开扩展设置页面

### 3. 配置 OpenAI 服务

在沉浸式翻译的设置中：

1. **选择翻译服务**：选择"OpenAI"
2. **填写配置信息**：

| 配置项 | 值 |
|-------|-----|
| **API Key** | 您的 ZenMux API Key |
| **自定义 API 地址** | `https://zenmux.ai/api/v1` |
| **模型** | `anthropic/claude-3-5-sonnet-20241022` 或其他支持的模型 |

### 4. 高级配置

#### 推荐模型选择

::: code-group

```json [高质量翻译]
{
  "model": "anthropic/claude-3-5-sonnet-20241022",
  "description": "最高质量翻译，适合专业文档"
}
```

```json [快速翻译]
{
  "model": "anthropic/claude-3-5-haiku-20241022", 
  "description": "快速响应，适合日常浏览"
}
```

```json [多语言专用]
{
  "model": "google/gemini-1.5-pro-latest",
  "description": "多语言支持优秀"
}
```

:::

#### 自定义翻译提示词

为了获得更好的翻译效果，建议使用以下自定义提示词：

```text
你是一个专业的翻译专家。请将以下文本翻译成{目标语言}，要求：
1. 保持原文的语义和语调
2. 使用地道的表达方式
3. 保留专业术语的准确性
4. 对于代码、品牌名等专有名词保持原样

要翻译的文本：
{原文}
```

## 功能特性

### 完全支持的功能

- **网页翻译**：实时双语对照显示
- **PDF 翻译**：保持原始格式的 PDF 翻译
- **输入框翻译**：文本框内容实时翻译
- **鼠标悬停翻译**：快速查看单词或句子翻译
- **自定义样式**：调整翻译显示样式

### 增强功能

通过 ZenMux 获得的额外优势：

- **多模型选择**：可根据需求切换不同翻译模型
- **智能路由**：自动选择最优可用的模型服务
- **成本优化**：统一的计费方式，避免多平台分别付费
- **高可用性**：多供应商冗余，确保服务稳定性

## 常见问题

### Q: 翻译速度较慢怎么办？

**A:** 可以尝试：
- 切换到响应更快的模型（如 `claude-3-5-haiku-20241022`）
- 检查网络连接状况
- 确认 ZenMux 服务状态

### Q: 翻译质量不理想？

**A:** 建议：
- 升级到更高质量的模型（如 `claude-3-5-sonnet-20241022`）
- 调整自定义翻译提示词
- 针对特定领域使用专门的提示词模板

### Q: 某些专业术语翻译错误？

**A:** 解决方案：
- 在自定义提示词中添加术语表
- 使用针对特定领域训练更好的模型
- 建立个人术语库供参考
