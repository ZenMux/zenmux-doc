# Studio-Video

Studio-Video 是 ZenMux 平台提供的视频生成工作台，支持参考图生成、首尾帧生成、模型能力参数配置、任务历史、结果预览与下载，并支持订阅模式和按量付费模式。

不同视频模型支持的生成模式、比例、分辨率、时长和声音输出能力不同，页面会根据当前模型自动启用或禁用对应选项。

![Studio-Video 主界面截图](https://cdn.marmot-cloud.com/storage/zenmux/2026/05/28/Bg1Gn90/20260528172015.png)

## 快速开始

1. 访问 [Studio-Video](https://zenmux.ai/platform/video)
2. 根据需要切换 **订阅模式** 或 **按量付费模式**
3. 选择视频模型
4. 选择生成模式：**参考图生成** 或 **首尾帧**
5. 按需要上传参考图、首帧或末帧
6. 输入视频提示词
7. 设置比例、分辨率、时长和声音
8. 按 `Enter` 或点击发送按钮开始生成
9. 在任务卡片中查看生成进度，完成后点击视频预览或下载

从模型详情页进入 Studio-Video 时，页面会自动带入所选模型。链接中也支持使用 `model` 参数指定模型。

## 界面结构

Studio-Video 主要由三个区域组成：

| 区域         | 说明                                                         |
| ------------ | ------------------------------------------------------------ |
| 顶部计费入口 | 切换订阅模式或按量付费模式                                   |
| 任务区       | 展示生成历史、任务状态、结果视频和任务操作                   |
| 输入区       | 选择模式、模型、上传素材、输入提示词、配置生成参数并提交任务 |

任务区会自动滚动到最新任务。历史任务按时间分组，向上滚动可以加载更早记录。

## 计费方式

在开始生成前，先确认当前使用的计费方式。

![计费方式切换截图](https://cdn.marmot-cloud.com/storage/zenmux/2026/05/28/BnpkZpq/20260528172119.png)

| 模式         | 说明                             |
| ------------ | -------------------------------- |
| 订阅模式     | 使用订阅计划覆盖的模型额度       |
| 按量付费模式 | 使用账户余额，按实际生成消耗计费 |

如果当前模型不在订阅计划内，订阅模式会不可用，并提示切换到按量付费。选择更高等级订阅模型时，页面会提示升级订阅；使用按量付费但余额不足时，页面会提示充值。

## 选择模型

点击输入区工具栏中的模型名称可以打开模型选择器。

![视频模型选择器截图](https://cdn.marmot-cloud.com/storage/zenmux/2026/05/28/RzEwshw/20260528172342.png)

模型选择器会筛选出支持视频输出的模型，并支持：

- 搜索模型名称
- 查看模型详情、能力和可用状态
- 区分免费模型和高级模型
- 根据当前计费方式判断模型是否可选

模型会决定后续可用的生成模式、比例、分辨率、时长和声音选项。切换模型后，Studio-Video 会自动刷新这些选项。

## 选择生成模式

Studio-Video 当前支持两种生成模式：

![生成模式选择截图](https://cdn.marmot-cloud.com/storage/zenmux/2026/05/28/6Schh1p/20260528172236.png)

| 模式       | 适用场景                                   |
| ---------- | ------------------------------------------ |
| 参考图生成 | 使用一张或多张参考图，引导模型生成视频内容 |
| 首尾帧     | 指定视频开头帧和结尾帧，让模型补全中间变化 |

如果当前模型不支持某个模式，该模式会置灰不可选。

参考图生成模式下，可以先只输入提示词，也可以上传参考图来增强画面控制；具体是否需要图片，以当前模型能力和生成结果为准。

## 上传素材

### 参考图生成

选择 **参考图生成** 后，输入区左侧会显示参考图上传入口。

![参考图上传截图](https://cdn.marmot-cloud.com/storage/zenmux/2026/05/28/NoOzQJn/20260528172307.png)

- 支持上传多张参考图，数量上限以当前模型能力为准
- 已上传图片会以堆叠缩略图展示
- 鼠标悬停可展开图片列表，继续添加或移除图片
- 上传中的图片会显示进度，上传失败会显示失败状态

支持的图片格式：

| 类型 | 格式                                     |
| ---- | ---------------------------------------- |
| 图片 | `.jpg`、`.jpeg`、`.png`、`.gif`、`.webp` |

### 首尾帧

选择 **首尾帧** 后，输入区左侧会显示 **首帧** 和 **末帧** 上传入口。
