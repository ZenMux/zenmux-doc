# 数据服务（Data Services）

数据服务是 ZenMux 的核心数据管理开关，控制平台是否在服务端存储和处理您的请求数据。启用后可解锁 AI 保险赔付、请求详情查看、Chat 跨端同步等高级功能；关闭后，您的账户将进入 **ZDR（Zero Data Retention，零数据保留）模式**，所有数据仅保存在本地浏览器中。

::: tip 隐私优先
ZDR 模式适合对数据隐私有严格要求的用户。关闭数据服务后，ZenMux 不会在服务端保留您的请求内容和对话记录，最大程度保护您的数据隐私。
:::

## 数据服务解锁的功能

启用数据服务后，您可以使用以下高级功能：

| 功能           | 说明                                                         |
| -------------- | ------------------------------------------------------------ |
| **AI 保险赔付** | 当 API 请求因上游故障导致失败时，系统自动评估并提供赔付补偿 |
| **请求详情查看** | 在 Logs 页面查看每个 API 请求的完整详情，包括请求参数、响应内容、耗时和 Token 用量 |
| **Chat 跨端同步** | Chat 对话记录保存在云端，支持在多个设备和浏览器之间无缝同步 |
| **视频任务同步** | Video Studio 中的视频生成任务保存在云端，可跨设备查看任务状态和结果 |

## ZDR（零数据保留）模式

当数据服务关闭时，您的账户进入 ZDR 模式。在此模式下：

- **请求数据不上传**：API 请求的内容不会存储在 ZenMux 服务端
- **对话仅存本地**：Chat 会话保存在浏览器的本地存储（IndexedDB）中
- **视频任务仅存本地**：Video Studio 生成任务仅保存在当前浏览器
- **高级功能不可用**：AI 保险赔付、请求详情查看等依赖服务端数据的功能将无法使用

::: warning 数据存储风险
ZDR 模式下，对话记录和视频任务仅存储在当前浏览器中。清除浏览器数据、更换浏览器或设备将导致这些数据**永久丢失**，且无法恢复。请谨慎操作。
:::

## 配置方式

### 开启或关闭数据服务

1. 访问 [ZenMux 策略设置页面](https://zenmux.ai/settings/strategy)
2. 找到页面顶部的 **Data Services** 配置项
3. 通过开关切换启用或禁用状态
4. 切换后立即生效

## 工作原理

### 会话存储机制

数据服务的开关状态决定了**新创建的会话**的存储方式：

- **数据服务开启时**创建的会话 → 存储在云端，支持跨端同步
- **数据服务关闭时**创建的会话 → 仅存储在本地浏览器

::: details 重要：会话存储模式不可变更

每个会话的存储模式在**创建时确定**，后续不会随数据服务的开关状态变化：

- 在 ZDR 模式下创建的本地会话，即使之后重新开启数据服务，**仍然保持本地存储**
- 在数据服务开启时创建的云端会话，即使之后关闭数据服务，**仍然保持云端同步**

这意味着切换数据服务状态不会影响已有会话的存储方式，只影响后续新建的会话。

:::

### 会话类型识别

在 Chat 侧边栏中，您可以通过会话图标区分存储方式：

| 图标 | 含义 |
| ---- | ---- |
| ☁️ 云朵图标 | 云端同步会话，数据保存在服务端 |
| 🖥️ 桌面图标 | 本地会话，数据仅保存在当前浏览器 |

### API 调用

对于 API 调用（非 Chat）：

- **数据服务开启**：请求详情记录在 Logs 页面，可用于调试和分析；AI 保险基于请求记录进行赔付评估
- **数据服务关闭**：请求正常路由和响应，但不记录详情，Logs 页面无相应数据

## 常见问题

### Q: 关闭数据服务后，API 调用会受到影响吗？

A: 不会。关闭数据服务仅影响数据存储和高级功能，**不影响 API 的正常路由和响应**。您的 API 请求仍会照常处理，模型兜底、供应商路由等核心功能不受影响。

### Q: 切换数据服务状态后，已有的 Chat 会话会怎样？

A: 已有会话不受影响。每个会话的存储模式在创建时确定，后续不会改变。关闭数据服务只会让**新创建的会话**变为本地存储，已有的云端会话仍然正常同步。

### Q: ZDR 模式下的本地数据存储在哪里？

A: 数据存储在浏览器的 IndexedDB 中。这意味着数据与特定的浏览器绑定，清除浏览器数据、使用无痕模式或更换设备都会导致数据丢失。

### Q: 如何将本地会话迁移到云端？

A: 目前不支持将已创建的本地会话迁移到云端。如果您需要跨端同步，请先开启数据服务，然后创建新会话。

### Q: 关闭数据服务后能否使用 AI 保险？

A: 不能。AI 保险赔付依赖服务端的请求记录来评估赔付条件。在 ZDR 模式下，由于没有服务端数据留存，保险功能无法正常工作。

### Q: 数据服务开关会影响计费吗？

A: 不会。无论数据服务是否开启，API 调用的计费方式保持一致。数据服务仅控制数据存储和高级功能的可用性。
